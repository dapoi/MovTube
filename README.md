# Movie Catalogue Using Movie DB API
## Tech Stack
- Single activity using jetpack navigation component
- MVVM
- Paging 3
- Splashscreen API
- Kotlin DSL
- BuildSrc
- Retrofit
- Reactive programming using flow
- Hilt
- Moshi
- Youtube player
- Safe args
- Livedata
- Unit test
- Room database

## Note
```
Video player doesn't work on emulator, please use real device
```
